## Notes
- The application requires Right-to-Left (RTL) layout support out of the box. The root element will use `dir="rtl"`.
- Typography relies on the 'Almarai' Google Font for a modern, clean Arabic aesthetic.
- Forms and data fetching use standard React Query patterns as defined in the API manifest.
- Ensure all Shadcn UI components are available in `client/src/components/ui`.
